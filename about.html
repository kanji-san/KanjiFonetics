    О проекте

    Kanji-world - это веб-приложение, основанное на JavaScript и предназначенное для анализа чтений китайских иероглифов исходя из фонетика в их составе. Анализ возможно проводить для китайского, японского и корейского языков, а также кантонского диалекта китайского языка.

    Предполагается учет внутренней вложенной структуры фонетика. Для этого введены понятия элементарного и производного фонетика (см. раздел ФОНЕТИЧЕСКИЕ СЕМЬИ).

    В настоящее время основная работа заключается в поиске фонетиков и внесении их в базу данных. Просмотр базы фонетиков реализован в разделе ПОИСК ПО ФОНЕТИКУ. Поскольку часть фонетиков отсутствует в стандартных шрифтах, то для решения этой проблемы был создан собственный шрифт Fonetics (см. раздел ШРИФТ ФОНЕТИКОВ). Проект также использует стороннюю открытую базу данных Unihan для получения основных сведений о иерглифе. Поиск по базе Unihan реализован в разделе ПОИСК В UNIHAN.

    После окончания стадии поиска фонетиков (либо параллельно с ней) планируется реализовать несколько алгоритмов для визуализации полученных данных (подробнее см. разделы КАРТА ФОНЕТИКОВ и ФОНЕТИЧЕСКИЕ СЕМЬИ).

    На данный момент поиск фонетиков завершен примерно на 25%.
